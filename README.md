# Python Hash Cracker

A simple hash cracking tool written in Python for educational purposes.

## Features

- Dictionary attack
- Supports MD5 / SHA1 / SHA256
- Uses RockYou wordlist
- Progress bar
- Simple CLI interface

## Usage

python3 hashcracker.py

Enter hash: 5f4dcc3b5aa765d61d8327deb882cf99
Enter algorithm (md5/sha1/sha256): md5
